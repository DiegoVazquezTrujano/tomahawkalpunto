Genera un arreglo de numeros aleatorios donde los valores pueden repetirse.

  • Asigna valores al azar utilizando el tamańo solicitado como base para el limite de los numeros generados. Muestra por consola el proceso de asignacion.
  • Parameters
    arregloPuntero al arreglo donde se guardaran los numeros generados.
    nReferencia a la variable que almacena el tamańo actual del arreglo (se actualiza a tam).
    tamEl numero de elementos que se desean generar.

Genera un arreglo de numeros aleatorios asegurando que ningun valor se repita.

  • Utiliza un arreglo auxiliar (pool de numeros) para extraer valores aleatorios y evitar colisiones. Muestra por consola el proceso de asignacion.
  • Parameters
    arregloPuntero al arreglo donde se guardaran los numeros unicos.

    tamEl numero de elementos que se desean generar.
    MValor maximo opcional para el rango de numeros a generar (por defecto es 0).
